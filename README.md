# 📊 Sales Data Analysis

A data analysis project built using Python to explore supermarket sales data, identify business trends, and generate actionable insights through visualization.

---

## 🔍 Project Overview

This project performs end-to-end exploratory data analysis (EDA) on a supermarket sales dataset. It covers data cleaning, feature engineering, revenue analysis, and visual reporting to support data-driven decision making.

---

## 🛠️ Tech Stack

| Tool | Purpose |
|---|---|
| Python | Core programming language |
| Pandas | Data loading, cleaning, manipulation |
| NumPy | Numerical computations |
| Matplotlib | Data visualization |
| Seaborn | Statistical plots |
| Jupyter Notebook | Development environment |

---

## 📁 Dataset

- **File:** `SuperMarket Analysis.csv`
- **Contents:** Supermarket transaction records including date, product line, and sales figures

---

## 📌 Key Features

- **Data Cleaning** — Parsed and formatted date columns; created derived month feature
- **Revenue Analysis** — Calculated total revenue generated across the dataset
- **Daily Sales Trend** — Plotted day-wise sales to identify peak periods
- **Monthly Sales Analysis** — Bar chart comparison of sales performance across months
- **Best-Selling Product Lines** — Ranked product categories by total revenue
- **Top 5 Sales Days** — Identified the highest revenue-generating days

---

## 📈 Sample Visualizations

- Daily Sales Trend (Line Plot)
- Monthly Sales Analysis (Bar Plot)
- Best Selling Product Line (Horizontal Bar Plot)

## 🚀 Getting Started

### Prerequisites
```bash
pip install pandas numpy matplotlib seaborn
```

### Run the Notebook
```bash
git clone https://github.com/dhruvvv2904/Sales-Data-Analysis.git
cd Sales-Data-Analysis
jupyter notebook sales_analysis.ipynb
```
